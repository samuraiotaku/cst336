What is Game Development?


Game Development involves taking the design specifications of the game designer and put it together with art, sound, putting it together into a playable game.

It is a process that requires a lot of math, logic, and computing skills and tend to use the programming language C, C++, and C#.

Some of the things Game Programmers do include mapping terrain of the game environment, implementing artificial inteligence, and connecting the controller to the character on screen.
